## About

**Open-Mushaf Native** is a modern, minimalist Quran Mushaf built for seamless reading and Tafseer exploration.  
It emphasizes performance, offline access, and a distraction-free experience, powered by React Native and Expo.

### Key Features

- **Swipeable Navigation** â€“ Move between pages smoothly with gestures.  
- **Offline Access** â€“ Cache Quran pages and Tafseer data for full offline use.  
- **Dynamic Tafseer Popups** â€“ Resize Tafseer sections with drag gestures.  
- **Optimized Image Loading** â€“ Uses `expo-image` for efficient memory handling.  
- **Lightweight State Management** â€“ Powered by **Jotai** atoms.  
- **Cross-Platform Ready** â€“ One codebase for Android, iOS, Web, and macOS.

## Installation

### Requirements
- Node.js **v22+**
- Expo SDK **52+**

### Steps
```bash
git clone https://github.com/adelpro/open-mushaf-native.git
cd open-mushaf-native
npm install
npx expo start
```
